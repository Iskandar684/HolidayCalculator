Инструкция по разворачиванию сервера.

1. Переменную JAVA_HOME указать на jdk8 
	-для Linux:
	JAVA_HOME=/opt/jdk1.8.0
	export JAVA_HOME
	PATH="$JAVA_HOME/bin:$PATH"

2. Скачать WildFly (Jboss) версии 20.0.0.Final из сайта:
        http://wildfly.org/downloads/    

3. Распаковать HolidayCalculator-user-server.zip в  /wildfly-10.1.0.Final/
